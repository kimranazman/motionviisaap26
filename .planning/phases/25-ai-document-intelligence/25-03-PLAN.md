```typescript
{
  success: true
  project: { id, revenue }
  document: { id, aiStatus, aiAnalyzedAt }
```

<task type="auto">
  <name>Task 2: Create receipt import API route with category creation</name>
  <files>src/app/api/ai/import/receipt/route.ts</files>
  <action>
Create src/app/api/ai/import/receipt/route.ts:

POST handler that accepts ReceiptExtraction data and creates cost entries.

Request body:
```typescript
{
  projectId: string
  extraction: ReceiptExtraction
  items: {
    description: string
    amount: number
    categoryId: string | null  // null if new category needed
    suggestedCategory?: string // AI-suggested category name (used when categoryId is null)
    include: boolean    // Whether to import this item
  }[]
}
```

Implementation:
1. Require editor role
2. Validate projectId exists
3. Validate documentId exists and belongs to project
4. Filter items where include is true
5. For each included item:
   - If categoryId is null BUT suggestedCategory is provided:
     - Check if category with that name already exists (case-insensitive)
     - If exists, use existing category ID
     - If not exists, create new CostCategory with suggestedCategory name
     - Use the resulting category ID for the cost entry
   - If categoryId is provided, validate it exists
   - Create Cost entry with:
     - projectId
     - description
     - amount
     - categoryId (resolved from above logic)
     - date: extraction.receiptDate or now()
     - (Optional: link to document via notes or future field)
6. Update document:
   - Set aiStatus to 'IMPORTED'
   - Set aiAnalyzedAt to now()
7. Return success with created costs and any new categories

Response:
```typescript
{
  success: true
  costsCreated: number
  costs: Cost[]
  categoriesCreated: { id: string, name: string }[]  // New categories created from AI suggestions
  document: { id, aiStatus, aiAnalyzedAt }
}
```

Error handling:
- 404 if project or document not found
- 400 if document doesn't belong to project
- 400 if document category is not RECEIPT
- 400 if categoryId provided but invalid (skip category creation logic)
  </action>

Test: POST with categoryId=null and suggestedCategory="New Category"

  </verify>
  <done>Receipt import API creates cost entries from AI extraction, can create new categories from AI suggestions when categoryId is null, and marks document as imported.</done>

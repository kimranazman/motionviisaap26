## Goal
Companies page and Suppliers page have consistent spacing matching the Price Comparison page layout.

## Context
The Price Comparison page (`src/app/(dashboard)/supplier-items/page.tsx`) uses `<div className="space-y-6 p-6">` which provides both vertical spacing between sections AND padding around the content. The Companies and Suppliers pages only have `<div className="space-y-6">` (no padding), resulting in content touching the edges.

## Tasks

<task id="1">


This adds `p-6` (1.5rem / 24px) padding on all sides, matching the Price Comparison page.
